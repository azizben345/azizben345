Reflection

Throughout the whole semester of the course SECV-2223 (Web Programming), I have been assigned to some tasks related to the course:
1. Assignment 1: Report
2. Assignment 2: Website
3. Assignment 3: GitHub e-Portfolio
4. Project
After receiving comments from the lecturer, I have reflected on my submitted assignments and project.

In Assignment 1: Report, I conducted a comparative evaluation of 5 e-commerce websites using Google Lighthouse. From this assignment, I learnt to evaluate critical aspects of websites, such as performance and accessibility. By comparing websites side by side, I can identify key strengths and weaknesses in their web design and functionality. These findings can serve as a guide for the developers to improve their web platforms and enhance their overall efficiency.
For improvements, I suggests to incorporating more visual aids like charts and graphs could further enhance the clarity and impact of the report.

In Assignment 2: Website, I created a personal website landing page using HTML and CSS Grid layout. From this assignment, I learnt to effectively organize the webpage into distinct sections: header, left and right menus, content, and footer, ensuring a logical flow and easy navigationof the website. With the provided theme samples as references, they helped in replicating a professional and aesthetically pleasing design, which includes welcome header, navigational menu and copyright notice. CSS played an important role to ensure not only the visual, but also the responsiveness across devices. Overall, this assignment not only reinforced my understanding of HTML and CSS Grid but also underscored the importance of user-centered design and personalization in web development.
For improvements, I suggests more freedom given to students on how to structure the website, not restricting to certain format. I understand this is to test certain topic uniformly, but it is better to give them space to explore their creativity.

In Assignment 3: GitHub e-Portfolio, which is this particular assignment, I created an e-portfolio using the platform GitHub, implementing web programming. From this assignment, it was an enriching experience which allows me to showcase my academic journey in an organized, digital format. It also tested my overall understanding and proficiency in HTML, CSS and JavaScript language. Building the e-portfolio on GitHub consolidated my understanding of version control and repository management. It was gratifying to see all my work presented in one cohesive platform. This not only only helped me reflect on my accomplishments but also highlighted areas for future growth.
For improvements, I suggests that students should update the e-portfolio regularly, even outside of the assignment, as it can serve well in a career pursuing and growth.

In the Project, we made a web-based hostel damage complaint system. Working on the project provided practical experience in applying theoretical knowledge to real-world problems. It tested our whole proficiency in HTML, CSS, JavaScript, PHP, MySQL, and also in hosting website locally and using assisting software(GitHub and localhost000). Not only it honed my problem-solving skills and fostered creativity and innovation, it also enhance my team skills, such as communication and collaborations.
For improvements, I suggests the tutorial for the website hosting is more in-depth and updated, as the suggested software may be updated from time to time.
